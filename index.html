How to use

  1. Click Start on an event lane to begin timing — or Start All to launch every lane together.
  2. Add more events with the Add Event button and rename each lane to match what you're timing.
  3. Tap Lap to record a split; the lap list shows total time and the gap since the last lap.
  4. Use Pause/Reset per lane, or Stop All / Reset All to control everything at once.

FAQ

Can I time more than one thing at the same time?

Yes — that's the point. Add a lane per event, name them, and run them in parallel. You can start and stop each independently or use Start All / Stop All to control them together.

How accurate is the stopwatch?

It uses your browser's high-resolution clock (performance.now) and updates smoothly, so it's accurate to hundredths of a second for normal use. Background tabs may throttle updates slightly, but elapsed time stays correct.

What are lap or split times?

A lap records the elapsed time at the moment you tap it without stopping the clock. The lap list shows each lap's total time and how long it was since the previous lap.

Does it keep running if I refresh or close the tab?

Your events and their times are saved locally in your browser, so a refresh restores them and a running stopwatch keeps counting. Clearing your browser data will reset them.

Is anything uploaded?

No. The stopwatch runs entirely in your browser. No account, no server — your data never leaves your device.
